## Setup
Both programs can be run using the same virtual environment
Clone the repo, then `cd assignment-02-Alphazerfall`.

The same virtual environment is used for both programs. Set it up using `python -m venv .venv`.

Activate the virtual environment using `.venv\Scripts\activate` on Windows and `source .venv/bin/activate` on Linux/Mac.

Install the required dependencies via `pip install -r requirements.txt`
